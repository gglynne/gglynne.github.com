ruby hash notation

This:

options = {} 
options[:open_now] = 
true options[:filename] = 'jeff.txt' 
options[:lines_to_read] = 50

is equivalent to:

options = { :open_now => true, :filename => 'jeff.txt', :lines_to_read => 50 }

haml syntax

html:

<strong class="code" id="message">Hello, World!</strong>

haml equivalent:

%strong{:class => "code", :id => "message"} Hello, World!

i.e.

%<tag><ruby hash> <content>

where the keys of the hash are symbols.

class and id attributes are common so can be further abbreviated:

%strong.code#message Hello, World!

i.e.

%<tag>.<class>#id <content>

div tags are common so omitting the tag defaults to thos:

so:

.content Hello, World!

expand to:

<div class='content'>Hello, World!</div>

summary:

%strong{:class => "code", :id => "message"} Hello, World!
<strong class='code' id='message'>Hello, World!</strong>

%strong.code#message Hello, World!
<strong class='code' id='message'>Hello, World!</strong>

.content Hello, World!
<div class='content'>Hello, World!</div>
