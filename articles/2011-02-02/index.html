Standard markdown syntax1 does not support footnotes, but fortunately pandoc2 does.

block quotes

use email indents:

this is a block quote

verbatim

use tab indents:

this is verbatim
text

note: the first tab indent is ignored, so tab it again to indent it:

    this is verbatim
    text
